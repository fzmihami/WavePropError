# WavePropError

WavePropError is a tool designed to evaluate linear wave propagation errors for various numerical schemes, flux reconstruction methods, and grid resolution.

For a full description of the project, including installation instructions, usage details, and examples, please refer to the [official documentation](https://waveproperror.readthedocs.io).
