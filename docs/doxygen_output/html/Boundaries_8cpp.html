Detailed Description

Implementation of functions for applying the sponge-layer boundary condition in the numerical wave solver to absorb incoming waves.

This file contains the implementation of the ApplySpongeLayer function, which applies a sponge layer to the left and right boundaries of the computational domain. The sponge layer is used to absorb incoming waves and prevent reflections from the boundaries.

void ApplySpongeLayer ( State & S1)

Applies the sponge-layer boundary condition to the left and right boundaries of the computational domain. This fucntion is based on the approach described in Larsen and Dancy (1983).

This function modifies the following variables in the State object S1 for grid points inside the sponge layer:

  • S1.Eta: The water surface elevation.
  • S1.Hn: The total water depth.
  • S1.Un: The horizontal velocity.
  • S1.Pn: The conserved momentum in Nwogu's equation.
Parameters
S1Reference to the State object containing wave field variables at the current time step t.
