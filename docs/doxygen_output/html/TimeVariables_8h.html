Detailed Description

Implements the State class and related functions for managing time-dependent variables in the numerical wave solver.

This file defines the State class, which stores the primary state variables (e.g., water height, velocity) at each time step. It includes methods for updating the state and handling memory allocation. Additional functions manage simulation progress reporting, record time series data at gauge points, and save results in binary format.
