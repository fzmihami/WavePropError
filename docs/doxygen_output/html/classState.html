Detailed Description

Represents the state variables of the numerical wave solver.

The State class contains the state variables (i.e., water height, velocity, etc.) at the current time step. It includes methods for updating the state and managing memory.

◆ Update()

void State::Update ( State & other)

Swaps the pointers of two State objects.

This function swaps the pointers of the state variables between two State objects, effectively updating their values. This is useful for updating the state variables at each time step without copying the data.

Parameters
otherReference to another State object to swap with.
