Detailed Description

Implements functions for solving the conservative form of Nwogu's equations.

This file provides the implementation of solver routines for Nwogu's equations, supporting both staggered and collocated grid configurations. It includes logic for handling various time integration methods and numerical flux schemes.

Solves the Nwogu's equation using a collocated grid for the first step of the Runge-Kutta time integration.

This function computes the fluxes and source terms for the continuity and momentum equations, and updates the state variables accordingly. It is the first step in the Runge-Kutta time integration.
