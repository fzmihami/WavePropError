Detailed Description

Computes the fluxes for the Shallow Water Equations in a staggered grid.

The flux_staggered class is responsible for computing the fluxes in the shallow water equations using a conservative staggered scheme (Stelling and Duinmeijer, 2003). This flux computation is based on second-order reconstruction.
