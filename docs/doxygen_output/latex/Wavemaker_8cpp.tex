\doxysubsection{Detailed Description}
Implements the \doxylink{classWaveMaker}{Wave\+Maker} class for generating wave forcing through a source term. 

This file provides the implementation of the \doxylink{classWaveMaker}{Wave\+Maker} class, which computes the source term used to generate waves within the computational domain. It also manages and stores the relevant wavemaker parameters. The wave forcing is constructed by superimposing contributions from multiple frequency components.
