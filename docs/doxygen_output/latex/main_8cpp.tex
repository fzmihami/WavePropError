\doxysubsection{Detailed Description}
Main file for the numerical wave solver. 

This file contains the main functions and logic for the numerical wave solver. It initializes the model parameters, sets up the grid and time variables for either staggered or collocated grids, and runs the time-\/stepping loop for the simulation.

that initializes the run parameters using the iput steering file, defines the grid and time variables for either staggered or collocated grids, sets up the flux, dispersion, and wavemaker objects, and runs the time-\/stepping loop. saves the simulation results to binary files, which will be read and plotted using the python scripts.

The main components of the code include\+:
\begin{DoxyItemize}
\item {\bfseries{\doxylink{classInput}{Input} Handling}}\+: Reads user input parameters from the steering file created by the python script.
\item {\bfseries{Grid Definition}}\+: Sets up the grid based on the selected scheme (staggered or collocated).
\item {\bfseries{Time Variables}}\+: Allocates and initializes time variables for the simulation. This includes the free surface elevation and velocity.
\item {\bfseries{Flux Computation}}\+: Computes the SW fluxes based on the selected numerical scheme (HLLC, HLL, central upwind or upwind for the conservative staggered scheme).
\item {\bfseries{\doxylink{classDispersion}{Dispersion}}}\+: Implements and discretizes the dispersion terms based on the selected grid type.
\item {\bfseries{Wavemaker}}\+: Generates wave forcing terms based on the approach described in wei et al. (1999).
\item {\bfseries{Time Integration}}\+: Solves the governing equations using Runge-\/\+Kutta methods (1st, 2nd, or 3rd order).
\item {\bfseries{Output}}\+: Saves time variables to binary files for post-\/processing and visualization.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
The code uses Open\+MP for parallelization and requires the Open\+MP library to be linked during compilation. 
\end{DoxyNote}
