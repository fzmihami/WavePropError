\doxysubsection{Detailed Description}
Implements functions for solving the conservative form of Nwogu\textquotesingle{}s equations. 

This file provides the implementation of solver routines for Nwogu\textquotesingle{}s equations, supporting both staggered and collocated grid configurations. It includes logic for handling various time integration methods and numerical flux schemes.
