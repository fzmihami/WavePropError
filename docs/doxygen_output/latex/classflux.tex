\doxysubsection{Detailed Description}
Computes the fluxes for the Shallow Water Equations in a collocated grid. 

The {\ttfamily flux} class is responsible for computing the fluxes in the shallow water equations using Finite Volume Godunov\textquotesingle{}s method. It includes different reconstruction orders and flux computation schemes. The class also computes the wave speed for the CFL condition.
