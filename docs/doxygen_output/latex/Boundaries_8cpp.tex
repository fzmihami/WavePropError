\doxysubsection{Detailed Description}
Implementation of functions for applying the sponge-\/layer boundary condition in the numerical wave solver to absorb incoming waves. 

This file contains the implementation of the {\ttfamily Apply\+Sponge\+Layer} function, which applies a sponge layer to the left and right boundaries of the computational domain. The sponge layer is used to absorb incoming waves and prevent reflections from the boundaries.

Applies the sponge-\/layer boundary condition to the left and right boundaries of the computational domain. This fucntion is based on the approach described in Larsen and Dancy (1983). 

This function modifies the following variables in the {\ttfamily \doxylink{classState}{State}} object {\ttfamily S1} for grid points inside the sponge layer\+:

\item {\ttfamily S1.\+Hn}\+: The total water depth.
\item {\ttfamily S1.\+Un}\+: The horizontal velocity.
