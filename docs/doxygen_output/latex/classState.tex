\doxysubsection{Detailed Description}
Represents the state variables of the numerical wave solver. 

The {\ttfamily \doxylink{classState}{State}} class contains the state variables (i.\+e., water height, velocity, etc.) at the current time step. It includes methods for updating the state and managing memory.
