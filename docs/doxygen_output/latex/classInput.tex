\doxysubsection{Detailed Description}
Represents the user-\/defined input parameters for the numerical wave solver. 

The {\ttfamily \doxylink{classInput}{Input}} class contains methods for reading user-\/defined parameters from an input file, including simulation parameters, grid properties, wave data, and scheme properties. It also includes a method for logging the input parameters to the console.
