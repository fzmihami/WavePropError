\doxysubsection{Detailed Description}
Defines the \doxylink{classDispersion}{Dispersion} class for computing dispersive terms in the numerical wave solver. 

This file contains the implementation of the \doxylink{classDispersion}{Dispersion} class, which handles the calculation and application of dispersive terms in both the continuity and momentum equations. It also includes a method for solving a tridiagonal system using the Thomas algorithm to compute the velocity field.
