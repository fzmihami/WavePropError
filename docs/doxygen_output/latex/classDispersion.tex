\doxysubsection{Detailed Description}
Stores and computes dispersive terms for the numerical wave solver. 

The {\ttfamily \doxylink{classDispersion}{Dispersion}} class calculates and applies dispersive terms in the continuity and momentum equations. It also provides functionality for solving a tridiagonal system using the Thomas algorithm.
